# Asset Directories

Place your brand assets in the following directories:

## Logos
`/assets/logos/`
- logo-primary.svg
- logo-primary.png
- logo-secondary.svg
- logo-secondary.png

## Gallery Images
`/assets/gallery/`
- photo-1.jpg
- photo-2.jpg
- etc.

## Fonts (optional)
`/assets/fonts/`
- For custom fonts hosted locally
- Reference them in data.json with type: "file"

All paths in data.json should start with `/assets/...`
